# Backend Architecture

The CTF FlagFrenzy backend provides the APIs and services that power the platform.

## Technology Stack

- Node.js with Express
- TypeScript
- PostgreSQL database
- Redis for caching and real-time updates

## Core Services

- **Authentication Service**: Manages user identity and access control
- **Challenge Service**: Handles challenge deployment, status, and flag validation
- **Scoring Service**: Calculates points and maintains the competition scoreboard
- **Team Service**: Manages team creation, updates, and member coordination
- **Notification Service**: Provides real-time updates and announcements
