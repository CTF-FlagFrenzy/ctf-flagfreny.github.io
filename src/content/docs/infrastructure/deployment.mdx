# Deployment

CTF FlagFrenzy is designed for easy deployment on Kubernetes clusters.

## Prerequisites

- Kubernetes cluster (K3s recommended for smaller deployments)
- Helm for package management
- kubectl configured for your cluster
- Domain name with DNS pointing to your cluster

## Installation Steps

1. Clone the repository:
   `ash
   git clone https://github.com/CTF-FlagFrenzy/deployment.git
   cd deployment
   `

2. Configure the values in \alues.yaml\:
   `ash
   cp values.example.yaml values.yaml
   # Edit values.yaml with your configuration
   `

3. Deploy with Helm:
   `ash
   helm install ctf-flagfrenzy ./chart
   `

4. Verify the deployment:
   `ash
   kubectl get pods -n ctf-flagfrenzy
   `

## Configuration Options

### Environment Variables

- DB_HOST: PostgreSQL host
- DB_PORT: PostgreSQL port
- DB_USER: Database username
- DB_PASSWORD: Database password
- REDIS_HOST: Redis host
- REDIS_PORT: Redis port
- JWT_SECRET: Secret for JWT tokens
- ADMIN_EMAIL: Default admin email
- ADMIN_PASSWORD: Default admin password

### Resource Scaling

You can adjust resource allocations in the \alues.yaml\ file:

`yaml
services:
  api:
    replicas: 3
    resources:
      limits:
        cpu: 1
        memory: 1Gi
      requests:
        cpu: 500m
        memory: 512Mi
  frontend:
    replicas: 2
    # ...
`
